# SDVCymraeg

Cyfieithiad Cymraeg o "Stardew Valley". Gwaith ar fynd, efallai byth i orffen. Dim linc "Nexus" tan bod mwy o waith wedi'i wneud.

Mae'r prosiect yma'n prosiect personol, felly does dim llawer o siawns bydd "pull requests" yn cael ei derbyn. Os ydych chi'n gweld camgymeriad, agor "issue". Peidiwch â defnyddio LLMau neu "generative AI" i gyfieithu neu prawfddarllen y prosiect.

-----

Welsh translation of Stardew Valley. Work in progress, maybe never to be finished. No Nexus link until more work has been done.

This is a personal project, so there's not much of a chance of pull requests being accepted. If you see a mistake, open an issue instead. Don't use LLMs or genAI to translate or proofread the project, especially if you don't speak Welsh.
